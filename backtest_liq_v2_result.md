## LIQ v2 신호 정의

```
ATR Capitulation: 비정상적 변동성 + 거래량 폭발 + 하한 꼬리 흡수 + 양봉 확인

조건 (4가지 AND):
  1. 캔들 크기 (high - low) > ATR(14) × atr_mult
  2. 거래량 > MA20_volume × vol_mult
  3. 하한 꼬리 비율 (close - low) / (high - low) > wick_ratio
  4. 양봉 마감 (close > open)
```
