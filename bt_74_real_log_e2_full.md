## 7) 결정 권장

### 가중 평가

- **실데이터 (1개월, BULL 후반)**: E2 net edge 음수 → 완화 가치 있음
- **백테스트 (4년, BULL+BEAR)**: E2 net edge 양수 (CAGR +27.73%) → 유지 가치 있음
- **결정**: 가중 평균 / 부분 완화 검토

### 권장 옵션 (우선순위)

**1순위 — gap 임계값 완화 (-10% → -5%)**:
- 현재 EMA200 gap = -8% 근처 → -10% 임계값에서 차단
- -5% 임계값이면 차단 해제 → 24건 차단 중 절반 이상 진입 가능
- BEAR 시 -10% 이상 빠질 때만 차단 유지 → 보호 효과 살림

**2순위 — bars 540 (3개월 → 1.5개월) 완화**:
- 차단 기간 짧아짐 → BULL 회귀 빠른 캐치
- 단 백테스트 결과 (#65) 에서 미검증 — 별도 검증 필요

**3순위 — 신호 강도 기반 동적 사이즈**:
- 강신호 (xgb≥0.65, score≥5): 60% 사이즈로 E2 예외 진입
- 약신호: 차단 유지
- 백테스트 (#71/#72 결과) 와 정합 — 단계 진입 가능

**기각 옵션**:
- E2 완전 OFF: 백테스트 -30% 위험 (2022 BEAR 재발 시 회복 불가)
