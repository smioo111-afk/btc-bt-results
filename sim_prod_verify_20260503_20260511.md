## 7. 발견된 차이

### 7-1. AI Gate (영향: 낮음)
- **실봇**: Static 0.58 OR Dynamic p60(last 20)
- **시뮬**: 모든 v20.x 백테스트에서 `patch_ai_gate_off` → Gate 미적용
- **본 기간 영향**: 0건. 실봇 dyn 경로로 PASS, 시뮬도 PASS. 결과 동일.
- **잠재 영향**: 실봇이 dyn FAIL했을 시 시뮬은 진입 (false positive). 본 기간에는 미발생.

### 7-2. E2 Block (영향: 없음)
- **실봇 v20.9.15**: `E2_ENABLED=False` (v20.9.10 #74 데이터 누적 모드)
- **시뮬**: cfg.e2_block_mode 옵션. v20.9.15 정합 백테스트는 `E2_ENABLED=False` 설정 필요.
- **본 기간 영향**: 일봉 가격대 118~120M, EMA200 ~121M → F2 활성 가능. 시뮬을 F2 ON으로 돌리면 실봇과 다르게 진입 차단됨.
- **조치**: v20.9.15 정합 백테스트는 반드시 `E2_ENABLED=False` 또는 동등 cfg 설정.

### 7-3. 인트라캔들 하드스톱 (영향: 0.03%)
- **실봇**: 매분 시세 폴링 → 저가 도달 즉시 시장가 매도 (슬리피지 발생)
- **시뮬**: 봉 저가에서 stop level 트리거 → stop level 그대로 청산 (슬리피지 없음)
- **본 기간**: −0.029% 시뮬이 우호적. 통계적으로 무시 가능.

### 7-4. Kill Switch (영향: 없음)
- **실봇**: MDD-based + PF-based 두 트리거 (v20.8)
- **시뮬**: passive MDD 추적만, 트레이딩 중단 logic 없음
- **본 기간**: MDD 약 −0.7% (단일 트레이드), KS 미발동. 영향 0.

### 7-5. 섀도우와의 불일치 (시뮬과 무관)
- 섀도우는 v20.9.4 기준 코드로 동작 → would_enter 무효
- xgb_prob·score·regime 1차 출력은 모두 일치, 단지 **결정 로직만 구버전**
- 시뮬 정합성 검증과는 별개 이슈
