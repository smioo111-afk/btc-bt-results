## BS 신호 정의

```
BS = higher_low ∩ fake_filter ∩ body_confirm
  higher_low    = low[i] > low[i-3]
  fake_filter   = close[i] > max(high[i-10..i-1]) × 1.002
  body_confirm  = close[i] > open[i]
```

BS 발동률: **5.13%** (전체 6898봉 중 354봉)
O3 발동률 (참고): 1.44% (99봉)
