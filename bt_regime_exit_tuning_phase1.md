## 9. 구조적 발견

1. **"수익구간 다녀왔다 손실 전환" = regime 문제 아닌 전략 구조 문제**: TU/Range/Volatile 모두 손실 MFE 1.5~1.9%, MFE≥1% 50~56%로 유사. TU가 전용 로직을 가지고도 동일 패턴.
2. **Volatile은 이미 "트레일링 주력" 구조**: 9건 중 8건 트레일링, ATR 1건. 전용 파라미터 도입 여지 없음.
3. **Range의 계단손절(평균 +6.52%)은 이미 "Range 전용 exit 역할" 수행 중**: 회복기 Range n=10 중 계단 3건, 트레일링 3건, ATR 4건으로 exit 경로 다변화됨.
4. **회복기 수익 저조의 원인은 "exit 로직"이 아닌 "entry 규모/타이밍"**: 회복기 Range+Volatile 9건 손실 중 6건이 MFE 0~1.4%로 애초에 수익 구간 자체가 좁았음.
